## How to Run

### Option 1: Using npm
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3000`

### Option 2: Using yarn
1. Install dependencies:
   ```bash
   yarn
   ```

2. Start the development server:
   ```bash
   yarn dev
   ```

3. Open your browser to `http://localhost:3000`

## How It Works

1. Click on "Taboushi" to reveal the Valentine question
2. Try clicking "No" - you'll see a funny popup!
3. After the popup, the "No" button will run away from your cursor
4. The "Yes" button becomes active after the first "No" attempt
5. Click "Yes" to answer the dancing question
6. The "No" button on the dancing popup also runs away!
7. Click "Yes" to see fireworks! ğŸ†
8. Finally, see the "Save the Date" for February 20!
